若颠木之有由蘖

发表于 2024-01-28 本文字数 1.1k 字 阅读时长 1 分钟
  • Homepage for EkkoXiao
首页

“沉舟侧畔千帆过，病树前头万木春。”2022 年卡塔尔世界杯葡萄牙被摩洛哥淘汰之时，铺天盖地的舆论口诛笔伐似有如巨石压卵之势，摧克了多少人的心理防线；当最后阿根廷人捧起大力神杯之时，一切似乎尘埃落定。大势已去，又岂可强为，一切似乎都是天意的眷顾，厚此薄彼实为常理，空余嗟叹。

很长一段时间内，我身边 C 罗与葡萄牙🇵🇹的同好都少得可怜，不胜枚举者都趋附于梅西与阿根廷🇦🇷；纵观全网，这种显而易见的差异似乎存在于每个角落，以至于梅西球迷似乎总是自认为高人一等，颐指气使地大肆舌剑唇枪；当初究竟为何开始关注 C 罗，时日久远已难以谨记，或许正与梅西球迷的不良风气和跳脸行为脱不了干系。

针对此次香港事变，各大足球博主已经将其剖析得体无完肤，诸多论调众说纷纭，但都指向同一个结果：梅西在中国粉丝群体中除却远超过安菲尔德神迹程度的极大反转出现，已经沦为遭唾弃之敝履，难以翻身。此般充斥于互联网视频中的讥讽之语这里就不再赘述，更加令我迫不及待的便是亲眼目睹昔日张牙舞爪者今时不同往日的嘴脸。枯草逢春霖，整个寒假单单此一件东窗事发，便足以让我收获不尽的欢愉，但更多的还是欣慰与感慨。

克里斯蒂亚诺・罗纳尔多・多斯・桑托斯・阿维罗，若颠木之有由蘖，天其永我命于兹新邑，绍复先王之大业，厎绥四方。真正的领袖气质，堪以登神的自律与职业态度，极富个人魅力的人格，抛开其在足球领域的成就不谈，他便足以在芸芸众生中成为佼佼者。他任心而行，凡高大者无不蔑视，言语皆无过于本心，外界视其为狂妄不羁、自私自傲、口无遮拦；但他又待下充满温情，对于粉丝更是近乎宠溺，这正是一个有血有肉的，真正的人最好的证明。驰骋足坛近二十载仍心囿满腔热血，在充斥着权利与斗争的这番领域能够坚持自我与本心，绝不为违心之事，喜怒形于色而付诸于行，纵然遭贪腐之风盛行的足联及其鹰犬自媒体冷眼相待，那又何妨？远离虚名浮华的欧洲是非之地，或许在生涯的最后他已经给了我们答案。

初高中学业压力使得我对于 C 罗职业生涯诸多重大事迹及其真相都了解甚少，之于金球奖评选幕后、转会生涯等等也是才疏志短难堪落笔写下，故此文仅仅以一名普通粉丝的视角刻画出他心目中的葡萄牙超巨形象；希望未来的我，虽不必揽获如他一般璀璨的成就，惟愿能够在这错综复杂的世界中遵从自己的本心，觅得自己的真正所爱。

写到此处，不觉已到凌晨，临表涕零，似乎也仅仅是写了一些泛泛而谈的崆峒之言；今日单曲循环《大哉乾元》，将歌词与 C 罗的生平相联，又好似遥相呼应，“始见龙形汇，以天田冲腾直向九陲 “，“龙秀于象，引仙来访，诗蜀道河江 “，不胜感触，热泪盈眶；如有时日，也想做一曲混剪相献，但注定是难得闲暇了。

若我不失为 C 罗，弗格森爵士又安在呢？
